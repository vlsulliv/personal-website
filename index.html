Top Frameworks

visit github to see how I leverage these technologies.

  • HTML

    Technically HTML is a programming language; not a framework. In any case, I started my web developer journey with HTML. It is different from other languages because it is a declarative language. You "code" a multiple statements that "declare" what should exist and how they should be arranged on a web page. HTML is normally taught with "hands-on" method of teaching while ignoring concept. This is a missed opportunity to introduce classes and explain the transition from declarative to computation programming language.
    *Note: i'm speaking from my expirience after taking 2 programming bootcamps.

  • CSS

    Used in conjunction with HTML. Use selectors to reference the elements you (the developer) have created using HTML. Add rules to the elements you reference, that determine how the information should be displayed. Again, there is no computation - so CSS is also a declarative language. In general CSS adds "style" to web pages. I was suprised at the power of CSS - there is even pages created for the purpose of showcaing CSS art.

  • javascript

    One of the core technologies of the web - alongside HTML and CSS. Over 97% of the websites use JS on the client side to manipulate web page behavior.

  • JSON

    An open standard date interchange format that uses human-readable text to store and transmit data ovjects consisting of attribute-value pairs and arrays.

  • MongoDB

    A cross-platform document-oriented database program. MongoDB was the first database I was introduced to, and prefer it to MYSQL.

  • React

    Open-source front-end JS libtatyraryr lding user interfaces hased on UI components. It is maintaind by "Meta" and a community of individual devs.
